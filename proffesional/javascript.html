Introduction to JavaScript(JS)

JavaScript is a dynamic language which is mainly used for game designing. JavaScript(often shortened to JS) is a lightweight, interpreted, object-oriented language with first-class functions, and is best known as the scripting language for Web pages, but it's used in many non-browser environments as well. It is a prototype-based, multi-paradigm scripting language that is dynamic, and supports object-oriented, imperative, and functional programming styles.

Why is Javascript so popular?

We can make commands by ourselves in JavaScript as it is a very powerful dynamic language. It’s popular for building web apps and games because it’s the only language that is native to the web browser.
Here are some points that why 'JS' is so powerful-:
1. JS (Javascript) has the backing of some of the biggest IT industries.
2. Javascript can perform numerous taks which might not be possible for other languages easily.(Voice recognition, Text to Speech etc.)
3. Javascript runs on the browser, means it doesn't affect the server performance.
4. Since it doesn't run on the server, but on the browser, the process can be more event driven.
5. Javascript has backing of numerous libraries like JQUERY, node.js, react.js etc.
